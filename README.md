# Görüntü İşleme ile Sanal Davul
Bu proje, OpenCV kütüphanesi kullanılarak mavi bir nesne ile davul çalmanızı sağlar.

### Kullanılan Teknolojiler:
* Python 3
* OpenCV (Görüntü İşleme)
* Pygame (Ses Kontrolü)
* NumPy (Matris İşlemleri)

### Nasıl Çalıştırılır?
1. Kütüphaneleri kur: `pip install opencv-python numpy pygame`
2. Kodu çalıştır: `python davul.py`
3. Kameraya mavi bir nesne göster ve kutulara dokun!
